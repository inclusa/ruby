La paraula `yield` s'utilitza per cridar blocks definits, és una paraula genèrica per cridar troços de codi que han estat declarats.

```ruby
def callBlock
  yield
  yield
  yield
end

callBlock { puts "Contingut del block" }
```
Output

```ruby
Contingut del block
Contingut del block
Contingut del block
```
